## 🚀 Usage Instructions

### Local Development

1. **Clone/Download**
```bash
cd fraud-detection
```

2. **Install Dependencies**
```bash
pip install -r requirements.txt
```

3. **Run Application**
```bash
streamlit run app.py
```

4. **Access App**
```
Open browser: http://localhost:8501
```

### Model Training

1. **Prepare Data**
```bash
# Place dataset.csv in project directory
# Required: 30 features + Class column
```

2. **Run Training**
```bash
python train_model.py
```

3. **Generated Files**
```
- xgb_model.pkl
- xgb_scaler.pkl
- feature_columns.pkl
- model_evaluation.png
```

### Docker Deployment

```bash
# Build image
docker build -t fraud-detection .

# Run container
docker run -p 8501:8501 fraud-detection

# Or use Compose
docker-compose up
```
